# Star Wars - Search the Force

### `Version 1`
> User Stories
- [x] User shall be able to view all results at home page or with empty search term 
- [x] User shall be able to enter all or part of search terms
- [x] User shall be able to search by name of people
- [ ] User shall be able to search by name of planets
- [ ] User shall be able to search by name or model of spaceships
- [ ] User shall be able to search by name of model of vehicles
- [ ] User shall be able to search by title of films
- [ ] User shall be able to search by name of species

> Features
- [x] Search Bar
- [x] Results List
- [ ] Dropdown to select search space (*people, planets, films, etc*)
- [ ] Select Result Item to expand to additional information
